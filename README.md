# Triggerbox Deluxe

It's a trigger device for multiple devices and input counter.
- Raspberry Pi Pico
- 24V power input
- 8 x optocoupler input
- 8 x high side switch 24V 500mA output
- optional Wifi interface instead of USB connection (ESP32C3)
